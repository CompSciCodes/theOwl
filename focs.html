In the Fundamentals of Computer Science course, you will learn the basics of HTML5, which is the foundation language of web design and web development. This is the first step in learning how to code web apps. By the end of the course, you will be able to build simple websites.

CSS is short for Cascading Style Sheets, and it is the computer language that allows you to add color, design and layout to your websites. It is the styling language of web design and web development. Having a basic understanding of CSS is fundamental for any web developer.

Javascript is the programming language built into every web browser, and is the most popular programming language in the world. To take full advantage of HTML5, CSS and the web browsers capabilities, you need to learn at least basic Javascript.
